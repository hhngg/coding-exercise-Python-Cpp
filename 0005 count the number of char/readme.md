- Q5 Write a function CountRangeChars that take in a string parameter str, and 2 integer parameters start and end and a character parameter ch. The function counts the number of characters that are the same as ch that fall in the range of start and end and return it. For example, CountRangeChars ("AAABBBCCC", 2, 7, 'C') returns 2 as there are 2 'C' characters between positions 2 and 7.
